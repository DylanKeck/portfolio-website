Content Strategy

About Me: This section will introduce myself and highlight my background and soft skills.

Tech Stack: This section will showcase the languages, libraries and frameworks I know. I will do this by adding in each one with it respective badge.

Projects: This section will contain a slider that showcases my best projects with a short description and a list of technologies used.

Contact Form: This section will contain a contact form with input fields for name, email, and message so users can easily get in touch.

Footer: This section will have clickable icons that are linked to my socials where they can look at my LinkedIN or GitHub and a quicker way to email me directly.
