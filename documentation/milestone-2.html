Content Strategy

About Me: This section will introduce myself and highlight my background and soft skills. This section will also include a large image showing my skills at a quick glance along with establishing my role as an aspiring full stack developer.

Tech Stack: This section will showcase the languages, libraries and frameworks I know. I will do this by adding in each one with it respective badge.

Projects: This section will contain a slider that showcases my best projects with a short description and a list of technologies used. The user will be able to use arrow keys to switch between projects.

Contact Form: This section will contain a contact form with input fields for name, email, and message so users can easily get in touch. It will aslo include a small message about how I look forward to talking to them.

Footer: This section will have clickable icons that are linked to my socials where they can look at my LinkedIN or GitHub and a quicker way to email me directly.
